Draw Your Starter is a personal machine learning project, which aims to use a convolutional neural network to classify user-submitted drawings of starter Pokémon.

I am currently at the data gathering stage, so am welcoming your Pokémon, the worse the better! See my shameful efforts below.

In the future, I plan to build, train, and serve a model using Tensorflow, and potentially write a tutorial based on this, which may feature some lucky submissions.
